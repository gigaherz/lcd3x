LCD3x

The LCD simulation filter.

Download this project as a .zip file Download this project as a tar.gz file

What is LCD3x?

LCD3x is an image filter that simulates the aspect of an LCD screen while at the same time increasing the size of the source image. It is implemented using the Render Plugin Interface that the Kega emulator created, which is also compatible with the VisualBoyAdvance-M emulator.

How does it work?

The basic algorithm works by converting each pixel into a square 3x3 grid, where each column takes the Red, Green and Blue components of the pixel, respectively.

Because just taking the component results in a loss of 66% of the brightness, the current code takes also 75% of the other color components. This results in a total of a 83% of the original brightness, but it loses 75% of the effect's depth. This 75% value was chosen empirically because it seemed the best balance between brightness and effect depth.

There's two special versions of plugin, namely "VBlend" and "Scanlines". The former works on the top and bottom rows of each pixel's grid, and mixes 25% of the color content with the value of the row above or below. The Scanlines version does the same, but forgets to add the color to the bottom row, resulting in the bottom row of the grids to take a darker tone (75% of the original), giving it a scanline effect.

The download package contains all three versions, and it's up to the user to decide which one works best.
